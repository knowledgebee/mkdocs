一些乌七八糟的想法，算是读后感吧，上次更新这个网站的 Blog 已经是两年前了，突然更新是想着这个网站也没有人会注意到，，只要域名还没过期，就能一直在 Cloudflare 和 Github 上托管，没有什么成本，就当是随笔吧，看看几年或者更多年之后，还能保持现在的想法不能吧

<!--more-->

最近看了一些非天夜翔的小说，结局都是Happy Ending的，两个人在一起携手共进，虽然路途有困难，但相互之间的爱可以让两人在前进的路上披荆斩棘，最终到达幸福的彼岸。

从小说的虚拟世界回归生活的现实世界，回想起小说中所描写的幸福，自己仍然如同空白一样的感情经历，平凡的生活，枯燥的日子，没有如同干涸的土地一般，不禁渴望属于我自己的幸福。

虽然空虚渴望，但我仍然相信，我能够遇见属于我的爱，能够遇见属于我的幸福。还未遇到，只是时机未到，我会一直等，直到
